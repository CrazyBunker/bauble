# Balance Checker

Программа для проверки баланса через API yarurf.ru

## Требования
- Go 1.16+
- Файл конфигурации `config.yaml`

## Установка
```bash
make yarurf-balance-checker 
```
## Конфигурация

Создайте файл config.yaml:
```yaml
user:
  login: "ваш_логин"
  password: "ваш_пароль"
templates:
  high_balance: "Баланс: %.2f (всё хорошо)"
  low_balance: "Внимание! Баланс: %.2f (ниже нормы)"

cost: 36

notify:
  bot_token: "токен телеграмма"
  chat:
   - 1234567890
  time:
    hour: 12
    minute: 0
```

## Использование
###  Базовый запуск
```bash
./yarurf-balance-checker
```
### С указанием конфига
```bash
./yarurf-balance-checker -config /path/to/config.yaml
```
## Примеры форматов вывода

  - Если balance / config.Cost > 3 → high_balance шаблон
  - Если balance / config.Cost <= 3 → low_balance шаблон

    Баланс: 150.00 (всё хорошо)  # При остатке на балансе > 3 дней
    
    Внимание! Баланс: 80.00 (ниже нормы)  # При остатке на балансе < 3 дней

## Флаги

    -config - путь к файлу конфигурации (по умолчанию config.yaml)

## Логика работы
- Проверяет валидность текущей сессии по cookies
- Если сессия недействительна - выполняет авторизацию
- Делает запрос баланса
- Выводит результат в указанном формате в зависимости от того на сколько дней хватит текущего баланса
- Завершает работу
